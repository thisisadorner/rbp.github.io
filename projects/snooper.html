Bash Script | Started October 2025

View on GitHub →

About

Snooper is a command-line tool for searching through files for specific phrases. It supports wildcards and provides clear feedback on search results.

Why I Built This

I needed a quick way to search through log files without having to remember complex grep syntax every time. This tool makes it interactive and user-friendly.

What I Learned

  • The difference between quoted and unquoted variables in Bash
  • How glob expansion works
  • Proper error handling in shell scripts
  • Making scripts user-friendly with clear prompts and feedback

Current Features

  • Interactive prompts for search phrase and file pattern
  • Support for wildcards in file paths
  • Clear feedback on matches found
  • Error handling for invalid patterns

Future Plans

  • Add recursive directory searching
  • Case-insensitive search option
  • Display matching lines, not just file names
  • Color-coded output
  • Export results to a file

Technical Notes

The key challenge was understanding how Bash handles variable expansion. When you quote a variable containing wildcards, Bash treats them as literal characters. Removing the quotes allows glob expansion to work correctly.

Reflections

This was my first real Bash script project, and it taught me that even simple tools can be incredibly useful when they solve a specific problem you encounter regularly.
